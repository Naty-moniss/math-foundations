### 📌 Sobre o Projeto | À propos du projet

**PT-BR:** Este repositório foi criado para documentar minha jornada de estudos em matemática fundamental e avançada. O objetivo é aprofundar tópicos essenciais para a área de computação que não 
foram totalmente cobertos na grade curricular da faculdade, servindo como base técnica para minha transição acadêmica.

**FR:** Ce dépôt a été créé pour documenter mon parcours d'études en mathématiques fondamentales et avancées. L'objectif est d'approfondir des sujets essentiels pour l'informatique qui n'ont pas été
entièrement couverts par le programme universitaire, servant ainsi de base technique pour ma transition académique.

---

### 🎯 Objetivos | Objectifs

- [ ] **Base Teórica / Base Théorique:** Consolidar fundamentos de Álgebra, Cálculo e Lógica.
- [ ] **Aplicação / Application:** Conectar conceitos matemáticos com Análise e Desenvolvimento de Sistemas.
- [ ] **Preparação / Préparation:** Nivelamento para admissão em universidades francesas (L2/L3).
